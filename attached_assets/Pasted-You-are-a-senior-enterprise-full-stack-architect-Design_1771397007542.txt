You are a senior enterprise full-stack architect. Design and implement a production-grade CSI Student Chapter Management Platform with scalability, security, and clean architecture.

========================
PROJECT OVERVIEW
================

Build a dynamic web platform to manage CSI Student Chapter operations for a college department. The system must replace static websites with a secure, role-based, and scalable solution.

========================
TECH STACK
==========

Frontend:

* React (Vite) with TypeScript
* TailwindCSS
* React Router
* React Query (data fetching & caching)
* Axios

Backend:

* Node.js + Express with TypeScript
* MongoDB + Mongoose
* JWT Authentication (HTTP-only cookies)
* Zod/Joi validation

Security & DevOps:

* Helmet middleware
* Rate limiting
* bcrypt password hashing
* ESLint + Prettier
* Docker-ready setup
* Environment variables (.env)
